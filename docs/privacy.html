Privacy Policy · 隱私政策

Last updated · 最後更新：2026-05-26

Short version: Daily Six does not collect, transmit, or share any of your data. Everything lives in your own browser. We don't have a server to send it to.
簡短版：Daily Six 不收集、不傳輸、也不分享你的任何資料。所有資料都只留在你自己的瀏覽器裡，我們根本沒有伺服器可以接收它。

What Daily Six is · 這是什麼

Daily Six is a Chrome extension that helps you practice the Ivy Lee Method (six prioritized tasks per day, completed in order) from the Chrome Side Panel. It is a single-purpose productivity tool.

Daily Six 是一個 Chrome 擴充功能，協助你在 Chrome 側邊欄中實踐 Ivy Lee 工作法（每天挑出六件重要的事，依序完成）。它是單一用途的生產力工具。

What data Daily Six stores · 儲存哪些資料

The extension stores the following inside your own browser, using the chrome.storage.local API:

  • The tasks you create (text content you type)
  • The current day's list, plus historical days you have completed
  • Your settings (language preference, reminder times, retention window)
  • The completion timestamps of tasks you mark done

擴充功能使用 chrome.storage.local API，將以下內容儲存在你自己的瀏覽器裡：你建立的任務文字、當天的清單與你完成過的歷史紀錄、你的設定（語言、提醒時間、保留天數），以及你標記完成的時間戳記。

Where this is stored · 儲存位置： locally on your device, inside the Chrome profile you are using. It is bound to your browser; if you sign into Chrome Sync with chrome.storage.local syncing enabled at the browser level, Chrome itself may sync that data across your own devices. Daily Six does not initiate or control that sync. （資料只存在你裝置上的 Chrome 設定檔中。若你在瀏覽器層級開啟了 Chrome Sync，Chrome 本身可能在你自己的裝置間同步這些資料；Daily Six 不會發起或控制這個同步。）

What Daily Six does NOT do · 我們不做的事

  • It does not transmit any of your data to us, to any server, or to any third party. · 不會把你的資料傳給我們、任何伺服器或第三方。
  • It does not include any analytics, telemetry, error reporting, ads, or tracking SDKs. · 不含任何分析、遙測、錯誤回報、廣告或追蹤 SDK。
  • It does not request access to any web page contents. The extension declares no host permissions. It cannot read or modify the pages you visit. · 不要求存取任何網頁內容，也未宣告任何 host 權限，無法讀取或修改你瀏覽的頁面。
  • It does not require an account, sign-in, or any identifier. · 不需要帳號、登入或任何識別碼。
  • It does not share data with any service. · 不與任何服務分享資料。

Permissions used · 使用的權限

Permission · 權限 Why we need it · 用途
sidePanel To register and open the side panel that hosts the entire app. · 註冊並開啟承載整個應用程式的側邊欄。
storage To save your tasks and settings to chrome.storage.local on your device. · 將你的任務與設定儲存到裝置上的 chrome.storage.local。
alarms To schedule the daily reminders you opt into (all reminders are off by default until you enable them in Settings). · 排程你主動開啟的每日提醒（所有提醒預設關閉，需在設定中開啟）。
notifications To display those reminders as Chrome notifications. · 以 Chrome 通知顯示這些提醒。

No other permissions are requested. No host permissions are requested. · 不要求任何其他權限，也不要求任何 host 權限。

Cookies and tracking · Cookie 與追蹤

Daily Six uses none. · Daily Six 完全不使用。

Children · 兒童

Daily Six is suitable for all ages. We do not collect any data from any user, including children under 13. · Daily Six 適合所有年齡層使用。我們不收集任何使用者（包括 13 歲以下兒童）的任何資料。

Data export and deletion · 匯出與刪除

  • Export · 匯出： Settings → "Export JSON" downloads a snapshot of all stored data to a file you control. · 設定 →「匯出 JSON」會下載所有資料的快照成一個你掌控的檔案。
  • Delete · 刪除： Settings → "Reset all" wipes the extension's local storage on this device after a confirmation prompt. Uninstalling the extension also removes the data Chrome holds for it. · 設定 →「全部重設」會在確認後清除此裝置上的擴充功能本機儲存；移除擴充功能也會一併刪除 Chrome 為它保存的資料。

Changes to this policy · 政策變更

If the policy ever changes (for example, if a new permission becomes necessary), the updated version will be committed to this repository with a new "Last updated" date. The extension itself will continue to operate offline; we do not have a mechanism to push policy changes to you remotely. · 若政策有變動（例如需要新權限），更新版本會以新的「最後更新」日期提交到本儲存庫。擴充功能本身持續離線運作，我們沒有任何遠端推送政策變更的機制。
