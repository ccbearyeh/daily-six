關於這份原型

以下為 Chrome Side Panel（寬 400px）內各主要狀態的純靜態 HTML/CSS 畫面， 不含任何 JavaScript 互動邏輯。設計參考自 Ivy Lee Method 的「單點焦點」與 「強制取捨」原則 — 視覺刻意克制、留白多、不使用花俏的圖示，避免工具本身造成分神。 每一張對應 SPEC.md §7.2 描述的視圖。

畫面 1 · Planning 規劃階段 輸入今天最重要的 6 項任務、拖曳排序、鎖定並開始。 畫面 2 · Focus 焦點模式 Active Task 強調、後續任務鎖定、無法跳號完成。 畫面 3 · Rollover 跨日轉移對話框 新一天首次打開時提示，使用者確認要轉移的昨日未完成項。 畫面 4 · Reminders 主動提醒展示 Toolbar Badge 計數、桌面通知樣式，含三種情境對照。 畫面 5 · Settings (EN) 設定頁（英文） 展示 i18n 切換到 en 後的版本，含語系切換 UI。

原型限制

  • 所有畫面皆為靜態快照，按鈕、拖曳、勾選皆無動作。
  • 實際擴充功能將以 React + Vite 實作，狀態與互動見 SPEC.md §4–§9。
  • 顏色與字體採 design tokens 形式定義於 styles.css 頂部 :root。
  • Side Panel 在 Chrome 內寬度可由使用者拉動（280–600px）；原型固定 400px。
